# Command line converter
Going to do unit conversions

## Working features

#### Length converters
- [x] inch -> cm
- [x] cm -> inch
- [x] feet -> cm
- [x] cm -> feet
- [x] feet -> m
- [x] m -> feet
- [ ] mile (statute) -> km
- [ ] km -> mile (statute)

#### Temperature converters
- [x] fahrenheit -> celsius
- [x] celsius -> fahrenheit
